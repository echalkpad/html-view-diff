# 项目Project
- id, 字符串, 唯一标识
- name, 项目名称
- 需要划定范围(例如子域范围), 表示只检测范围的web, 和外链web 

# 版本Version
- 一个项目对应多个版本
- id, 字符串, 唯一标识
- recordDate, 收录日期

# 用例Case
- 一个项目对应多个用例
- 一个用例对应上一个用例
- id, 字符串, 唯一标识
- name, 名称, 唯一
- description, 描述, 对该用例能做什么的文本描述
- code, 脚本, 一段代码
- state, 状态, pass | fail | undefined
- on, 已经被用例执行引擎执行过
- active, 是否可用

# 快照Snapshot
- 一个用例对应多个状态
- id, 字符串, 唯一标识
- dom, 除去style属性的DOM
- model, 包括最终css和cssom的综合表示
- url, 也许需要?
- order, 顺序
